# Python Notes

## Variables
Used to store data

Example:
x = 10

## Data Types
int → whole numbers
float → decimal numbers
str → text
bool → True/False

## Operators
+  → addition
-  → subtraction
*  → multiplication
/  → division

## Loops
Used to repeat code

Example:
for i in range(5):
    print(i)

## Functions
Reusable block of code

Example:
def greet():
    print("Hello")
